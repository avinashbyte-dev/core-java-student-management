# Core Java Student Management System

This is a beginner-friendly Core Java console project built to strengthen Java fundamentals and OOP concepts.

## Features (Phase 1)
- Create student objects
- Display student details

## Concepts Used
- Classes and Objects
- Constructors
- this keyword
- Package structure

## How to Run
1. Clone the repository
2. Compile and run Main.java

## Learning Goal
To Understand deeply and build a strong Core Java foundation before moving to backend and full-stack development.
